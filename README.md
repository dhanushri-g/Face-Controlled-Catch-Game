# 🎮 Face-Controlled Catch Game

An innovative twist on the classic arcade-style catching game — controlled entirely using your **head movements**.

## 🚀 Overview

This project is an interactive and experimental game where balls fall from the top of the screen, and the player controls a horizontal bar (paddle) at the bottom to catch them.

What makes this game unique is its **hands-free control system**. Instead of using a keyboard or mouse, the paddle is controlled through **real-time face tracking**, allowing players to play the game simply by moving their head.

## 🧠 How It Works

* The game accesses your device’s camera to detect and track your face.
* Moving your head left or right moves the paddle accordingly.
* The objective is to catch as many falling balls as possible.
* No physical input devices are required — just look and move!

## 🎯 Features

* 👤 Real-time face tracking
* 🎮 Fully hands-free gameplay
* ⚡ Smooth and responsive controls
* 🧩 Simple yet addictive mechanics
* 💡 Demonstrates human-computer interaction using vision
